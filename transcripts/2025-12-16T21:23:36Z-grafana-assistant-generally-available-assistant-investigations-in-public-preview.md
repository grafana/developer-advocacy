In this video, the speaker provides an update on Grafana Assistant, highlighting its new features and capabilities. Key points include the introduction of customizable "rules" that allow users to tailor the assistant's behavior and advice based on their unique setups. The ability to connect to MCP servers, including popular platforms like GitHub and AWS, is discussed, allowing for seamless integration into workflows. Another notable innovation is the "infrastructure memory," where agents analyze telemetry to improve response times and accuracy in future interactions. The speaker announces that Grafana Assistant is now generally available at a pricing of $20 per active user per month starting January 26, with a reminder for current users to continue using the service throughout the year. The video concludes with a mention of the public preview of investigations and a promise of more features to come.

I am thrilled to give you an update on **Grafana Assistant**. We're constantly expanding its capabilities, and I want to highlight a couple of my favorite features that I think you might like too.

First, **rules** allow you to control how the assistant behaves and what advice it provides. You can share interesting details about the nuances in your setup, which the assistant will consider when addressing other problems. If you've spent a long time establishing the best practices unique to your case, you can incorporate that knowledge through these rules. The assistant will take this information into account whenever it answers questions or tries to help users. This gives you full control, as you can write these rules in natural language.

Additionally, you can now connect **MCP servers**. If you want to interact with third-party systems, GitHub has a robust MCP server, and several ticketing systems now offer them as well. I believe AWS has one too, and we expect to see more emerging over time. If there's something very specific you want to achieve with the assistant in your environment, you can build a custom MCP server and plug it in, making those tools available to the assistant. This means the assistant can take action and fully integrate into your workflows today.

Another cool innovation from the team is **infrastructure memory**. This feature allows agents to scour telemetry data and build a map of what they observe. This map helps the assistant answer questions more quickly in the future, as the agents will consult this memory when making decisions. You can enhance this functionality with the integration of the **knowledge graph**, which allows the assistant to learn and improve the more you use it.

I am very pleased to announce that **Grafana Assistant** is now generally available, and we will also be announcing the pricing: **$20 per active user per month** starting January 26. If you're already using it, please continue to do so for the rest of the year as a thank you. We genuinely hope you find as much value in it as we do.

Lastly, **Investigations** is now in public preview, and we have more exciting features coming soon!
